# 3D Solar System Lab

An interactive Solar System scene packaged as a single self-contained HTML
file. The page uses inline CSS and JavaScript, so it runs directly from
`index.html` without `node_modules`, a build step, or a local server.

## Features

- One-file canvas-based 3D solar system with perspective projection.
- The Sun, all eight planets, Earth's Moon, and Saturn's ring.
- Orbit focus list, click-to-select planets, and live planet metrics.
- Scale modes for balanced viewing, distance emphasis, and size emphasis.
- Speed slider, pause/play control, drag-to-rotate, wheel zoom, and camera
  presets.
- Responsive HUD layout for desktop and mobile screens.

## Run Locally

Open `index.html` directly in a modern browser.
